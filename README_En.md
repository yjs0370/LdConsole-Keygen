## What is Tianrui Green Shield?

Tianrui Green Shield Data Leakage Prevention System (referred to as the Data Leakage Prevention System) is a security product independently developed by Tianrui Co., Ltd. It uses driver-layer transparent encryption technology to achieve secure encryption of electronic files. This encryption system ensures data security and usage security from the source. It follows the idea of security protection based on the file lifecycle and integrates cryptographic technology, access control, and audit tracking to provide comprehensive protection for the storage, access, transmission, and processing of electronic files in enterprises and institutions.

Tianrui Green Shield is not free software. If you wish to use it, you should purchase a license from Tianrui Green Shield.

## Supported Versions
Only tested on **V6.00.200720.SC**, **V6.51.221108.SC**, and **V7.00.230704.SC**. The default login password for **V7.00.230704.SC** is **Admin@2006**.

## How to Use the Keygen

The user ID is set to **1111111111** by default.

The default registration code is **73B4C0ED7DD7BF5A03ABD5155DD25560**.

The registration code is only used during the installation of the Green Shield server and is calculated based on the user ID.

The authorization file is saved in the root directory of the keygen program by default, named **trInfoPf1.irs**.

After generating the `trInfoPf1.irs` file, place it in **C:\Program Files\Tipray\LeaderEIS\Server\Engine Service Program** (the default installation directory of Green Shield).

Restart the **LeaderServer** service or reboot the computer to complete the registration.

Each authorization module can allocate up to **65535** licenses.

The user count is **65535**.
